# Compilador Lukasiewicz

![build status](https://travis-ci.org/lucaspetry/lukasiewicz-compiler.svg?branch=master)

- Na pasta **projeto1**, utilize os seguintes comandos:

| Função                 | Comando      |
| :--------------------- | :----------- |
| Compilar               | `make`       |
| Limpar                 | `make clean` |
| Testar                 | `make test`  |
| Limpar/Compilar/Testar | `make all`   |
